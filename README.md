## Packages installation

```bash
$ yarn
```

## Running the app

```bash
# start development environment
$ yarn start:dev
```

## Building the app

```bash
# build dev environment
$ yarn build:dev

# build uat environment
$ yarn build:uat

# build prod environment
$ yarn build:prod

# build devprem environment
$ yarn build:devprem
```

## Checking for the code style exceptions

```bash
# eslint check
$ yarn lint:check
```
